HTML Forms Tutorial

HTML forms are a fundamental part of web development, allowing users to interact with web applications, submit data, and provide feedback.

Please remember the examples you see in this page are applied by my css. They will different if you try them.

Form Structure

A basic HTML form consists of the following components.

<form action="process.php" method="POST">
    <!-- Form elements go here -->
    <input type="text" name="username" id="username" placeholder="Username" required>
    <input type="password" name="password" id="password" placeholder="Password" required>
    <input type="submit" value="Submit">
</form>
  1. <form>: The form element that contains all form controls.
  2. action: The URL where the form data is sent for processing (typically a server-side script).
  3. method: The HTTP method (GET or POST) used to send data.

Form Controls

HTML offers various form controls for collecting user input.

  1. Text fields: <input type="text">
  2. Password fields: <input type="password">
  3. Textareas: <textarea></textarea>
  4. Radio buttons: <input type="radio">
  5. Checkboxes: <input type="checkbox">
  6. Select dropdowns: <select></select>
    7. Buttons: <input type="submit">, <input type="reset>, <button>

Labels and IDs

  1. Labels improve the accessibility and usability of your forms.
  2. hey should be associated with form controls using the for attribute and matching id attributes.
<label for="username">Username:</label>
<input type="text" name="username" id="username">

Placeholder Text

  1. Use the placeholder attribute to provide hints or examples for users within form controls
<input type="text" placeholder="Enter your name" name="name" id="name">

Required Fields

  1. Use the required attribute to make a field mandatory.
  2. The browser will prevent submission until all required fields are filled.

Validation

  1. You can use HTML5 input attributes like type="email", type="number", and pattern to enforce data formats and validate input.
<input type="email" name="email" id="email" required>
<input type="number" name="age" id="age" min="0" max="120">
<input type="text" pattern="[A-Za-z]{3}">

Fieldset and Legend

  1. To group related form controls, use the <fieldset> and <legend> elements
<fieldset>
    <legend>Personal Information</legend>
    <!-- Form controls go here -->
</fieldset>

Multiple-Choice

  1. You can create radio buttons, checkboxes, and select options to allow users to make multiple choices
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
<input type="checkbox" name="interest" value="sports"> Sports
<input type="checkbox" name="interest" value="music"> Music

<select name="country">
    <option value="usa">United States</option>
    <option value="canada">Canada</option>
</select>
Male Female Sports Music

File Upload

Use the <input type="file"> control to allow users to upload files

<input type="file" name="file_upload">
