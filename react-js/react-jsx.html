React JSX Tutorial

In this Tutorial, you are gonna learn React JSX in detail to get started with react.

Before we begin, let's set up our React Environment.

Step1: Install Node.js and NPM

  1. Node.js is a JavaScript runtime environment that allows you to run JavaScript code outside of a browser.
  2. NPM is a package manager for Node.js that allows you to install and manage JavaScript packages.
  3. To install Node.js and NPM, you can visit their websites and download the latest version for your operating system.
  4. To run our React code use the code editorVS Code (Recomonded)

Step2: Install Create React App

  1. Create React App is a tool that makes it easy to create new React projects.
  2. To install Create React App, run the below command in your terminal
    3. npm install -g create-react-app

Step 3: Create a new React project

  1. Once you have Create React App installed, you can create a new React project by running the following command in your termina
    2. npx create-react-app my-react-app
  2. This will create a new directory called my-react-app containing your new React project.

Step 4: Start the development server

  1. To start the development server, run the below command in your termina
    2. cd my-react-app
npm start

The development server will start on port 3000 by default. You can open http://localhost:3000 in your browser to view your React project.

Step 5: Edit your React project

  1. You can edit your React project by opening the src directory in your code editor.
  2. The srcdirectory contains all of the source code for your React project.

Step 6: Save your changes and the browser will automatically reload

  1. Once you have made changes to your React project, save your changes and the browser will automatically reload.
  2. This makes it easy to see the results of your changes immediately.

Step 7: Deploy your React project

  1. Once you are finished developing your React project, you can deploy it to production.
  2. Using: Vercel, Netlify, or Heroku

What is React JSX

  1. React JSX, which stands for JavaScript XML, is a syntax extension to JavaScript that allows developers to write HTML-like code within their JavaScript code.
  2. This makes it easier to create and maintain complex user interfaces, as developers can keep their HTML and JavaScript code in the same place.
  3. For example the below are the valid JSX Exmaples

Single Element JSX


const h1 = <h1>Hello, World!</h1>

Nested Elements

<div>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
    <ul>
        <li>Apple</li>
        <li>Orange</li>
        <li>Mango</li>
    </ul>
</div>

In the latest version or react we use Fragments<>Elements</> instead of <div> Element.

Rendering JSX into HTML

  1. We create JSX Elements to display as an html Element in the DOM.
  2. To do that we import react and ReactDOM from the package and we render JSX into webpage.
import react from "react"
import ReactDOM from "react-dom/client"

const h1 = <h1>Hello, World!</h1>
const container = document.getElementById("root");
const root = ReactDOM.createRoot(container);
root.render(h1);

After this, run the code and see the changes in the brower.

The above code will render the h1 and display in the index.html file under this <div id="root"></div>

When we see in console tab, we get the rendered h1 tag
