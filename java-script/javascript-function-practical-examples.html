JavaScript Functions Practical Examples

In this JavaScript Functions Practical Examples, you will get an idea of how we can use functions in real world applications.

Rock Paper Scissors Game

// Define the array of possible choices
const choices = ["rock", "paper", "scissors"];

// Define a function to generate a random choice
function randomChoice() {
    return Math.floor(Math.random() * choices.length);
}

// Define a function to get the human's choice
function humanChoice() {
    const ch = choices[randomChoice()];
    return ch;
}

// Define a function to get the computer's choice
function computerChoice() {
    const ch2 = choices[randomChoice()];
    return ch2;
}

// Define a function to play the game
function playGame() {
    const human = humanChoice();
    const computer = computerChoice();

    console.log(`Human: ${human}`);
    console.log(`Computer: ${computer}`);

    if (
    (human === "rock" && computer === "scissors") ||
    (human === "paper" && computer === "rock") ||
    (human === "scissors" && computer === "paper")
    ) {
    console.log("Human wins!");
    } else {
    console.log("Computer wins!");
    }
}

// Play the game
playGame();
/* Output: 
Human: rock
Computer: paper
Computer wins!
*/

playGame();
/*Output: 
Human: rock
Computer: scissors
Human wins!
*/
  1. The choices array defines the three possible choices for the game: rock, paper, and scissors.
  2. The randomChoice() function generates a random number between 0 and the length of the choices array. This number is then used to index into the choices array to return a random choice.
  3. The humanChoice() function prompts the user to enter their choice and then returns the user's input.
  4. computerChoice() This function returns the computer's choice. It does this by generating a random choice from the choices array and then returning that choice.
  5. playGame() This function plays a round of rock-paper-scissors. It does this by calling the humanChoice and computerChoice functions to get the human's and computer's choices, respectively. It then compares the human's and computer's choices to determine the winner. Finally, it prints the winner to the console.

User Profiles

const users = []; // Array to store user objects

// Function to add a user
function addUser(name, id, email) {
    users.push({
    name,
    id,
    email
    });
}

// Function to remove a user
function removeUser(userId) {
    const index = users.findIndex(user => user.id === userId);
    if (index !== -1) {
    users.splice(index, 1);
    }
}

// Function to search for a user
function searchUser(userId) {
    const index = users.findIndex(user => user.id === userId);
    return index !== -1;
}

// Example usage:

// Add a new user object
addUser("Alice", 1, "alice@example.com");

// Remove the user from the system
removeUser(1);

// Search for the user
const isUserPresent = searchUser(1);

console.log(isUserPresent); // false
  1. addUser(): This function takes three arguments: name, ID, and email. It uses these arguments to create a new user object and add it to the users array.
  2. removeUser(): This function takes one argument: userID. It uses this argument to find the index of the user object with the given ID in the users array. If the user object is found, it is removed from the users array.
  3. searchUser() function: This function takes one argument: userID. It uses this argument to find the index of the user object with the given ID in the users array. If the user object is found, the function returns true. Otherwise, it returns false.
